BeefTimer
Удобный таймер для спидранеров.
---
<h3>Установка</h3>

1. Закиньте плагин в папку plugins.
2. Настройте конфиг и перезапустите плагин. 

<h3>Команды</h3>
Для получения списка команд, введите /beeftimer help

<h3>Права</h3>
<p>- beeftimer.help</p>
<p>- beeftimer.stop</p>
<p>- beeftimer.start</p>
<p>- beeftimer.continue</p>
